HangMan Out

ABOUT THE GAME
  1. Hangman is a word-guessing game where you try to uncover a hidden word by guessing one letter at a time.
  2. You have a limited number of incorrect guesses before the game ends.
  3. Try to find out the meaningful word
HOW TO PLAY:
  1. Click on a letter to guess it.
  2. If the letter is correct, it will appear in the word.
  3. If the letter is incorrect, you'll lose one chance.
  4. You can guess until you either win or run out of chances.
